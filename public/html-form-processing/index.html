HTML Form Processing

This tutorial walks you through the creation of a CGI program that responds with the input values passed from an HTML form.

* This tutorial intends to walk you through the concept of CGI form processing and is designed to be langauge agnostic. Although this tutorial uses PHP as the server side programming language, the concepts can be coded in any language.

  • Create a CGI responder.
  • Test the script
  • Test program using cURL
  • CGI Input
  • Create HTML form

Create a CGI responder

Create a responder.php file on the website with the following code:

<?php
    print "Hello\n";
?>

Test the script

php responder.php

Test program using cURL

curl -k "http://__URL__/response.php"

The server should respond to both with the following:

Hello

CGI Input

<?php
    print $_REQUEST;
?>

Test program using a cURL GET request. On a *nix server command line enter:

curl -k "http://__URL__/response.php?key_1=value_1&key_2=value_2"

Test program using a POST request. On a *nix server command line enter:

curl -k "http://__URL__/response.php" -X POST -d "key_1=value_1&key_2=value_2"

Test program using a web browser. Enter the URL into the web browser:

http://__URL__/response.php" -X POST -d "key_1=value_1&key_2=value_2

The server should respond with the following:

Array

As appropriate a response as this is, let's see what the contents of the array are.

Update the program to print the contents of the request inputs.

<?php
    print json_encode($_REQUEST);
?>

The server should respond with the following JSON representation of the request inputs:

{"key_1":"value_1","key_2":"value_2"}

Since the request is an data structure we can also access the keys individually.

<?php
    print $_REQUEST['key_1'];
?>
